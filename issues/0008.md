## Description

Implement the short-link key generation logic in `internal/links/`. Keys are 6-character base-62 strings. Generation must handle collisions by retrying up to 5 times.

## Acceptance Criteria

- [ ] `GenerateKey() (string, error)` returns a cryptographically random 6-character base-62 string (`[a-zA-Z0-9]`)
- [ ] Uses `crypto/rand`, not `math/rand`
- [ ] `CreateLink(ctx, db, userID, destinationURL, title, expiresAt string) (*Link, error)` generates a key, checks for uniqueness in the DB, retries up to 5 times on collision, returns an error after 5 failures
- [ ] `CreateLinkWithAlias(ctx, db, userID, key, destinationURL, ...) (*Link, error)` accepts a user-supplied key; returns a conflict error if the key is already taken
- [ ] `go test ./internal/links/` passes for key format, uniqueness check, and collision-retry cases (use a mock DB interface)
