## Description

Implement the `seed` subcommand (`go run ./cmd/shortlinks seed`) that creates the initial admin user from `ADMIN_EMAIL` and a test link pointing to `https://www.wikipedia.org`. The command must be idempotent — safe to re-run without creating duplicates.

## Acceptance Criteria

- [ ] `go run ./cmd/shortlinks seed` (or compiled `shortlinks seed`) runs without error
- [ ] Creates a `users` row with `email = ADMIN_EMAIL`, `is_admin = true`, `active = true` — skips if the email already exists
- [ ] Creates a `links` row with a generated key, `destination_url = https://www.wikipedia.org`, `user_id` = the admin user's id, `active = true` — skips if a link with that destination already exists for the admin
- [ ] Prints the generated short URL: `Seed link: https://go.sstools.co/u/{key}`
- [ ] Idempotent: running twice produces no error and no duplicate rows
- [ ] Command reads config from `.env` / environment (uses the same config loader as the server)
