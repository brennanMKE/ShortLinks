## Description

Wire the Svelte production build into the Go binary using `//go:embed`. The catch-all `GET /*` handler serves the SPA's `index.html` for all paths not matched by an API or redirect route, enabling hard-refresh support.

## Acceptance Criteria

- [ ] `web/dist/` is populated by `npm run build` (run inside `web/`)
- [ ] `cmd/shortlinks/static.go` (or equivalent) contains:
  ```go
  //go:embed web/dist
  var staticFiles embed.FS
  ```
- [ ] `internal/handlers/static.go` serves the embedded FS:
  - Exact path matches (hashed assets like `assets/index-abc123.js`) served directly with `http.FS`
  - All other paths (`/`, `/anything`) return `web/dist/index.html` (SPA fallback)
- [ ] Static handler is the final route registered — after all `/u/`, `/api/`, `/auth/`, `/account/`, `/admin/`, `/health` routes
- [ ] `go build ./cmd/shortlinks` succeeds with `web/dist/` present; binary runs and serves the SPA
- [ ] `web/dist/` is listed in `.gitignore` (built artifact, not committed)
- [ ] `Makefile` (or `README` instructions) documents the two-step build: `cd web && npm run build && cd .. && go build ./cmd/shortlinks`
