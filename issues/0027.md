## Description

Implement the `/api/me` endpoint that returns the authenticated user's profile. Used by the Svelte SPA on load to determine whether to show the Admin tab.

## Acceptance Criteria

- [ ] `GET /api/me`: requires valid session (uses `RequireAuth` middleware)
- [ ] Returns `200 {"id": <int>, "email": "<string>", "is_admin": <bool>}`
- [ ] Returns `401` if not authenticated
- [ ] `go test ./internal/handlers/` covers: authenticated returns profile with is_admin field, unauthenticated returns 401
