## Description

Implement the Link Detail view (`web/src/views/LinkDetail.svelte`). Shows full stats for a single link including click count, UTM breakdown, and controls to copy the short URL or deactivate the link.

## Acceptance Criteria

- [ ] Loaded when `currentView === "link-detail"`; reads selected link from `selectedLink` store
- [ ] Calls `GET /api/links/{key}` to load full detail including `utm_stats`
- [ ] Displays: short URL (with copy-to-clipboard button), destination URL, title, created_at, expires_at, active status, total click count
- [ ] UTM breakdown section shows top sources, mediums, and campaigns in sorted lists (by count descending); skipped if all counts are zero
- [ ] "Deactivate" button calls `DELETE /api/links/{key}`; on success updates link in `links` store and navigates back to `"dashboard"`
- [ ] "Back" button navigates to `"dashboard"` without a reload
- [ ] Denied links show the denial reason label prominently
- [ ] Component is shown when `currentView === "link-detail"`

Dependencies: #0022 (link detail API), #0030 (UTM analytics in response), #0031 (stores, navigation).
